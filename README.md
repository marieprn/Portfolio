# Mon Portfolio

## Description

Ce projet est mon portfolio développé avec React.
Il présente mes compétences, mes projets et mes informations de contact.

## Fonctionnalités

* Présentation de mon profil
* Liste de mes projets
* Section compétences
* Formulaire de contact
* Design responsive (mobile / desktop)

## Technologies utilisées

* React
* JavaScript (ES6+)
* HTML5 / CSS3
* Sass

## Installation

Clone le projet :

```bash
git clone https://github.com/marieprn/Portfolio
```

Accède au dossier :

```bash
cd portfolio
```

Installe les dépendances :

```bash
npm install
```

Lance le projet :

```bash
npm run dev
```

## Utilisation

Une fois lancé, le site est accessible sur :
http://localhost:5173/
